# Jargon 翻譯對照詞庫

> 「翻譯員」voice 的核心工具：把 MBA / 顧問 / 技術 jargon 翻譯成普通人能聽懂的話。
>
> 使用原則：**寫稿時若用到左欄詞，必須改用右欄翻譯版**。Voice Lock 第 5 條會檢查左欄是否出現。

## 翻譯原則（寫稿時參考）

### 原則 1：用具體場景替代抽象概念

❌「**降低 NPI 週期**」
✅「**從想法到產品上架的時間從 6 個月變 6 週**」

### 原則 2：用日常比喻替代專有名詞

❌「**這個工具具備異質整合能力**」
✅「**這個工具能把不同 AI 模型組起來用**」

### 原則 3：用結果替代術語

❌「**提升企業的數位韌性**」
✅「**讓你的公司在 AI 時代撐得住**」

### 原則 4：用量化替代模糊形容詞

❌「**顯著提升營運效率**」
✅「**省下 30% 的人力時間**」

### 原則 5：保留戰略骨架，只換語言層

戰略思考（影響、成本、時機、競爭）可以保留，但**用詞要降級**。例：

| 戰略概念 | 顧問腔 | 翻譯員 |
|---|---|---|
| 影響 | 戰略意涵 | 對你公司的影響是 |
| 成本 | 總體擁有成本（TCO）| 總共要花多少錢 |
| 時機 | 戰略窗口期 | 現在做還是半年後做 |
| 競爭 | 結構性重塑 | 整個產業要重新洗牌 |

## 為什麼這個詞庫是 voice 的核心

**經營顧問** 跟 **技術翻譯員** 的差異不在「懂不懂戰略」，而在**表達層**。

兩個人可能腦中跑一樣的戰略邏輯（影響、成本、時機），但：
- 顧問用 jargon 寫 → 顯得專業 → 同行讀 → 但企業老闆讀不下去 → 失去 Page 受眾
- 翻譯員用人話寫 → 顯得親切 → 老闆讀得下去 → 願意分享 → Page 受眾擴大

**B2B FB Page 經營要的是後者**。LinkedIn / 學術論文要前者。場合不同。

## 自己擴充詞庫的方法

當你發現某個 jargon 你經常想用、但這份字典沒列：

1. 先用 jargon 寫一遍
2. 想像「**如果我跟一個沒念過 MBA 的工廠老闆解釋這個概念，我會怎麼說？**」
3. 把口語版寫下來
4. 加進這份字典的「延伸對照」段
5. 下次 routine 用 prompt inline 帶這條規則
