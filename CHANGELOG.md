# Changelog

## v0.1.0 — 2026-04-29 — Initial Release

First public release. Built from実戰 lessons of running a B2B FB Page from 5 followers.

### Core Concept

「**翻譯員 over 分析師**」: take MBA / consultant voice intent (strategic framing, business impact) but strip the jargon. Aimed at B2B Page operators whose readers are managers — people who want strategic insight in plain language, not academic prose.

### What's Included

- **3-mode schedule**：Mon/Wed/Fri = Tool Dive, Tue/Thu/Sat = News Commentary, Sun = Weekly Recap
- **3 formulas**：F-TOOL-DIVE, F-DAILY-NEWS, F-WEEK-RECAP
- **Word-count discipline**：350-400 (daily) / 800-1000 (Sunday)
- **Translation dictionary**：9 jargon terms → plain Chinese equivalents
- **Voice Lock 8 checklist**：unified pre-publish gate
- **L3.5 automation pattern**：cloud routine drafts + local human confirms (compatible with open-source SKILL.md safety gate)
- **Phase 1/2/3 strategy**：staged growth playbook for new pages

### Differences vs Upstream `social-post`

| Dimension | social-post | fb-page-translator |
|---|---|---|
| Audience | personal creator (<5K) | B2B / thought-leadership Page |
| Voice | viral creator (爽啦/草/！！！/emoji) | translator (formal-but-accessible, 0 emoji, 0 ！) |
| Schedule | daily uniform | 3-mode rotation |
| Word count | flexible 4-段 4-句 | strict 350-400 / 800-1000 |
| Formulas | F1-F13 viral | F-TOOL-DIVE / F-DAILY-NEWS / F-WEEK-RECAP |
| Automation | manual | L3.5 cloud routine + local confirm |
| Jargon | not addressed | translation dictionary |

### Known Limitations

- Designed for Chinese-language Pages. Voice rules are written for 中文 syntax.
- Open-source version cannot bypass the publish-confirmation gate (R8). For full automation, fork to a private version or use Meta Graph API.
- 5-follower validation pending — published Day 1 example post is in a real account.
